# Project Structure

The repository is organized as follows:

- `main.py`: The main script containing the complete workflow from data loading to model evaluation.
- `TABLE_OF_CONTENTS.md`: Table of contents for easy navigation.
- `INTRODUCTION.md`: Introduction to the project.
- `INSTALLATION.md`: Instructions on how to install dependencies and set up the project.
- `USAGE.md`: Instructions on how to run the project.
- `PROJECT_STRUCTURE.md`: Description of the project structure.
- `RESULTS.md`: Description and interpretation of the results.
- `LICENSE.md`: License information for the project.

This structure ensures a clear and organized layout, making it easy to navigate and understand the various components of the project.
